# Review Checklist

Before calling the MVP complete:

- [ ] No copyrighted assets from Castle Wars or Magic & Mayhem are used.
- [ ] Public-facing title is original.
- [ ] All 56 cards appear in UI.
- [ ] All cards use rethemed names but retain source/debug metadata.
- [ ] Exact starting stats: 2/5 for each resource engine, 30 Citadel, 10 Ward.
- [ ] Exact hand size: 8.
- [ ] Exact deck limits: 50–280 cards, max 5 copies.
- [ ] Exact turn options: cast 1, or discard up to 3, or end turn.
- [ ] Normal/bypass damage behaves correctly.
- [ ] Buffs consume correctly.
- [ ] Curse, Thief, Sabotage, Reverse, Roadblock work.
- [ ] Game can end by Citadel 100 or Citadel 0.
- [ ] Tests pass.
